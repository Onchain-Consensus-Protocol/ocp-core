# 承诺证明（POC）：一种生成昂贵社会共识的机制

## Proof of Commitment (POC): A Scheme for Costly Social Consensus

> **实现状态**：本文档定义 OCP 的机制层（POC）。共识上链的叙事框架见 [OCP 白皮书](./OCP_WHITEPAPER_CN.md)。

---

## 摘要

本文提出一种**三元结构**的链上协议，用于形成关于外部命题的、抗操纵的资本共识。协议通过三阶段实现：

1. **质押**：参与者以资本为信念加权，形成初始分布。
2. **锁定**：首次质押确定方向，资金不可撤回或换边，只能同侧追加。
3. **终局**：固定截止后，严格多数规则输出一次性最终共识。

该协议不生产真相，而是共识的**「压力测试熔炉」**，为预测、仲裁与治理提供无需信任的基础层。其底层逻辑即**承诺证明（POC）**：共识的权威性由资本的可验证锁定来证明。

**关键词**：质押、锁定、终局、承诺证明（POC）、比例结算、博弈论

## 1. 引言：共识的代价

### 1.1 问题

现有的去中心化系统如何对外部事实或主观选择做出**不可逆的、抗操纵的集体承诺**？

预言机依赖链下权威或复杂治理；链上法庭（如 Kleros）依赖多数投票；预测市场（如 Polymarket、Augur）依赖外部结算来源。共同问题是：**真相或终局裁决依赖外部裁判**，存在单点信任与操纵风险。

### 1.2 核心论点

最高形式的可信承诺，是**愿意为其承担巨大的、可验证的财务风险**。本协议旨在将这一原则代码化：不引入外部裁判，而让**资本本身**在固定截止的公开博弈中形成共识；该共识的价值来自资本被不可撤回地锁定并承担判断错误的损失。

### 1.3 协议哲学

**本协议不生产真相，只生产最昂贵的共识。** 你想用它来做什么——预测事件、仲裁争议、治理表决、传递信念——是应用层的事。

**本协议是一个通过质押、单侧锁定与固定截止，将资本承诺转化为终局结果的博弈机器。**

## 2. 机制设计：质押 — 锁定 — 终局

### 2.1 质押期

**质押**：个体将其资本在特定时间段内，单向绑定于某一命题之上的行为。这是信念的信号与成本。（*哲学隐喻：投入土壤的种子。是原料，但本身并非果实。*）

- **三态事件**：命题有三个互斥结果 YES、NO、INVALID（事实破裂/前提不存在/无法验证）
- **截止时间 $t_0$**：质押期结束并停止接收新资金的固定时刻
- **参与者**：任何持有可存入资产的地址，可向选项 YES、NO 或 INVALID 存入资金，**完全锁定**；每人仅能选择一方（可同侧加仓）

质押期形成公开的资本分布。YES 或 NO 只有在其本金严格超过 YES + NO + INVALID 总本金的 50% 时才构成二元多数。

### 2.2 单侧锁定与固定截止

YES、NO、INVALID 在同一个质押期内开放。地址首次质押后方向锁定，只能同侧追加，不能撤回或换边。这样，承诺不是可根据尾盘公开信息改写的免费期权。

截止时间在金库创建时确定，不因领先变化、资金流入或任何交易而延长。$t_0$ 到达后，任何地址可触发一次性终局。

### 2.3 共识形成阶段

**共识**：固定截止到达后，根据三侧本金与严格多数规则形成的终局状态。（*哲学隐喻：幸存者的奖杯。是最终产物，代表了截止时可验证的资本承诺。*）

- **终止条件**：固定质押截止时间到达
- **三态终局**：
  - YES > 总本金 50% → YES 胜出，YES 方按本金比例分配资金池
  - NO > 总本金 50% → NO 胜出，NO 方按本金比例分配资金池
  - 否则（包括刚好 50%）→ INVALID，所有参与者按本金比例取回资金

```
┌─────────────────────────────────────────────────────────────────┐
│  质押期（至 t₀）                                                 │
│  参与者向 YES、NO 或 INVALID 存入资金，完全锁定；每人至多一侧     │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  单侧锁定：首次质押后只能同侧追加，不可撤回或换边                 │
└─────────────────────────────────────────────────────────────────┘
                              │
                    固定截止 t₀
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  共识形成：YES>50% 或 NO>50% 通吃；否则 INVALID 全员退款；终局    │
└─────────────────────────────────────────────────────────────────┘
```

### 2.4 父子事件与历史

父子事件引用、自动退款和跨 Vault 参与者历史可在应用层构建，但当前核心 Vault 尚未实现这些能力，也不会用历史声誉改变质押权重或结算。

### 2.5 术语映射

| 旧表述 | 归属于 |
|--------|--------|
| 存入、承诺期、投票权重 | **质押** |
| 单侧、不可撤回、不可换边 | **锁定** |
| 比例结算、终局、INVALID 退款 | **共识** |

### 2.6 关键属性

| 属性 | 说明 |
|------|------|
| **极简** | 用户只有质押与领取；终局由任何地址触发 |
| **三态终局** | YES、NO 赢家通吃；INVALID 全员退款 |
| **一次性终局** | 协议定义，不可修改；避免无限博弈，共识权威性全球统一 |
| **无预言机** | 共识完全内生，不依赖外部数据源 |
| **最低质押额** | 金库发起人设定数值与单位 |

## 3. 理论分析：不可撤回承诺

### 3.1 博弈论模型

参与者选择方向 $s$ 与本金 $x$，并对自己判断正确的主观概率 $q$ 做出决策。若胜方资金为 $W$、可分配资金池为 $P$、资金锁定与交易成本为 $c$，其期望收益可抽象为：

$$
\mathbb{E}[U_{stake}] = q\left(\frac{x}{W}P-x\right) - (1-q)x-c
$$

不可撤回和不可换边使错误判断承担完整损失；最低质押额过滤过小交易。协议没有办法验证 $q$ 是否来自独立信息，因此结果只能解释为资本置信度，而非客观真理概率。

### 3.2 安全哲学：锁定是承诺的边界

- **不可撤回**：参与者不能在不利时取回本金。
- **不可换边**：参与者不能观察公开领先方向后免费追随。
- **固定截止**：任何人都不能通过交易延长博弈。
- **严格多数**：没有任何二元方向超过总本金 50% 时，协议选择 INVALID，而不是强行制造赢家。

### 3.3 透明资金流

全程透明让所有人都能审计资本分布，也会产生羊群和尾盘参与。协议接受这一取舍，不使用隐藏仓位或时间分层奖励制造表面上的早期活跃。改变终局需要投入足以改变总本金多数的资本。

### 3.4 与 Schelling Point 的对比

本协议是一个**资本化、不可撤回的 Schelling Game**：

| Schelling Game | OCP / POC 机制 |
|----------------|----------------|
| 投票权重 | 质押权重 |
| 协调成本 | 锁仓 + 判断错误损失 |
| 协调失败 | 直接资本损失 |
| 协调焦点 | 「截止时资本愿意承担哪个结果」 |

协调失败者承担真实资本成本。可视为 **Capital-Weighted Schelling Game with Fixed Finality**。

## 4. 协议属性与安全模型

### 4.1 无需预言机

共识完全内生。结算结果由链上资本分布决定，不依赖任何外部数据源或裁判。

### 4.2 最小化信任

只信任代码和参与者自身的逐利动机。无中心化管理员，无链下委员会。

### 4.3 安全边界

**明确承认**：当资本被系统性误导（如极端意识形态事件、信息不可公开验证、信号严重相关）时，协议输出的是**「昂贵错误」**而非真相。协议不声称生产真理，只声称生产**经高成本考验的共识**。适用域由应用层的事件选择规则界定（见第 5 节及应用规格文档）。

## 5. 应用场景：质押—锁定—终局的实例化

以下应用均为固定截止资本表决的具体实例化。**本节仅列举，详细机制见各应用规格文档**。

### 5.1 预测市场

完整的「质押（存款）— 锁定（不可换边）— 终局（结算）」循环：

- **质押**：用户买卖 YES/NO 份额、向金库存款，形成初始信念分布
- **锁定**：地址首次选择方向后只能同侧追加
- **共识**：固定截止后，严格多数形成二元结果，否则 INVALID

### 5.2 去中心化仲裁

- **质押**：原告/被告将抵押品存入 A/B
- **锁定**：原告、被告或 INVALID 方向的资金不可撤回或换边
- **共识**：截止后按严格多数形成终局裁决

### 5.3 重大治理投票

- **质押**：参与者对提案投票，资本锁定于赞成/反对
- **锁定**：赞成、反对或 INVALID 的资本承诺保持不可修改
- **共识**：截止时严格多数决定是否执行

### 5.4 信念信号与承诺

纯粹展示社区对某命题的信心强度。重要的是固定截止前不可撤回资本所显示的承诺硬度。

## 6. 讨论、局限与未来工作

### 6.1 参数影响

| 参数 | 设定方 | 过小 | 过大 |
|------|--------|------|------|
| 最低质押额 | 金库发起人 | 小额交易与噪音增多 | 参与门槛过高 |
| 质押期长度 | 应用层 | 信息聚合不足 | 资本锁定时间过长 |

### 6.2 局限

- **低流动性**：参与者少时，共识的「硬度」有限
- **高度模糊事件**：当信息不可获知或资本被系统性误导时，协议输出可能偏离真相
- **羊群与尾盘参与**：全程透明时，参与者可能观察公开分布后再决定是否下注

### 6.3 未来工作

- 跨链实现
- 身份关联、巨鲸影响与资本集中度研究
- 新应用形式（治理、仲裁的详细规格）

---

## 7. 结论

承诺证明（POC）提供了一种将**社会信念资本化并锁定**的基础工具。其输出是社会认知的**硬度的衡量**，而非绝对真理。

**本协议是一个将不可撤回质押转化为资本共识的博弈机器。**

比特币白皮书定义了「无需信任的价值转移」；本协议则定义「**无需信任的昂贵承诺**」——前者是货币的基石，后者是广义社会协调的基石。
